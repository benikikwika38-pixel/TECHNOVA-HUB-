📘 CHAPITRE 1 : PRÉSENTATION DU HTML

Définition

HTML est l’acronyme de HyperText Markup Language, ce qui signifie Langage de Balisage Hypertexte.

- C’est le langage standard utilisé pour créer et structurer le contenu des pages web.
- Ce n’est pas un langage de programmation : il ne fait pas de calcul, il décrit seulement la disposition et le sens des éléments.
- Son rôle : former le squelette de la page ; la décoration (couleurs, formes) se fera plus tard avec le langage CSS.

📘 CHAPITRE 2 : LES BALISES ET LES ATTRIBUTS

2.1 Définition d’une balise

Une balise est une instruction écrite entre les symboles < et >. Elle indique au navigateur quel type d’élément afficher.

2.2 Types de balises

- Balise avec contenu : S’ouvre et se ferme, elle encadre un texte ou un autre élément.

<p>Mon texte</p>

- Balise auto-fermante : N’a pas de contenu, elle fonctionne seule.

<br>
<img>

2.3 Définition d’un attribut

Un attribut est une information complémentaire ajoutée dans la balise pour préciser son fonctionnement.

<a href="page.html">Lien</a>

Dans cet exemple :
- href est l’attribut
- page.html est sa valeur

📘 CHAPITRE 3 : STRUCTURE GÉNÉRALE D’UNE PAGE HTML

<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <title>Titre de ma page</title>
</head>
<body>
  Ici tout le contenu visible
</body>
</html>

Définition des éléments :

- <!DOCTYPE html> : précise que le document utilise HTML5.
- <html> : balise racine qui contient toute la page.
- <head> : contient les informations invisibles pour l'utilisateur.
- <meta charset="UTF-8"> : permet d'afficher correctement les accents et caractères spéciaux.
- <title> : définit le nom affiché dans l’onglet du navigateur.
- <body> : contient tout ce que l’utilisateur voit à l’écran.

📘 CHAPITRE 4 : PRINCIPALES BALISES À UTILISER

<h1> à <h6> : Balises de titre du plus grand au plus petit.

<h1>Bienvenue</h1>

<p> : Balise de paragraphe.

<p>Ceci est un paragraphe.</p>

<strong> : Met le texte en gras.

<strong>Important</strong>

<em> : Met le texte en italique.

<em>À retenir</em>

<a> : Crée un lien.

<a href="contact.html">Contact</a>

<img> : Affiche une image.

<img src="photo.jpg" alt="Photo">

<ul> et <li> : Liste à puces.

<ul> <li>Point 1</li> </ul>

<ol> et <li> : Liste numérotée.

<ol> <li>Étape 1</li> </ol>

<!-- --> : Commentaire invisible.

📘 CHAPITRE 5 : RÈGLES DE BONNE ÉCRITURE

1. Toujours fermer les balises.

✅ <p>Texte</p>
❌ <p>Texte

2. Bien imbriquer les balises.

✅ <p><b>Bonjour</b></p>
❌ <p><b>Bonjour</p></b>

3. Mettre les attributs entre guillemets.

✅ href="page.html"
❌ href=page.html

4. Écrire les balises en minuscules pour une meilleure lisibilité.

✅ Conclusion : HTML est simple, il suffit de bien connaître les balises et de respecter la structure pour construire des pages web fonctionnelles.

🎨 DÉBUT DU COURS CSS

Niveau : Débutant
Prérequis : Bases de HTML

📘 CHAPITRE 1 : PRÉSENTATION DU CSS

Définition

CSS signifie Cascading Style Sheets, ce qui se traduit par Feuilles de Style en Cascade.

- C’est le langage utilisé pour mettre en forme et décorer les pages web créées en HTML.
- Rôle : Gère les couleurs, les polices, les tailles, les espacements, la disposition des éléments et l’arrière-plan.
- HTML représente la structure de la page.
- CSS représente l’apparence et la décoration.
- Un seul fichier CSS peut modifier l’apparence de tout un site web.

📘 CHAPITRE 2 : FONCTIONNEMENT ET SYNTAXE DE BASE

2.1 Comment fonctionne CSS ?

CSS agit en ciblant les éléments HTML pour leur appliquer des règles de style.

2.2 Syntaxe générale

sélecteur {
  propriété: valeur;
  propriété: valeur;
}

- Sélecteur : élément HTML à modifier.
- Propriété : ce que l’on veut changer.
- Valeur : le réglage choisi.
- Le point-virgule termine chaque instruction.

h1 {
  color: blue;
  text-align: center;
}

Tous les titres h1 seront bleus et centrés.

📘 CHAPITRE 3 : LES DIFFÉRENTS MOYENS D’UTILISER CSS

1️⃣ CSS en ligne

Le style est écrit directement dans la balise HTML.

<p style="color:red;">Texte rouge</p>

2️⃣ CSS interne

Le style est placé dans la balise <style> du document HTML.

<style>
body{background-color:lightgrey;}
h1{color:darkblue;}
</style>

3️⃣ CSS externe ✅

Méthode recommandée pour les vrais projets.

<link rel="stylesheet" href="style.css">

📘 CHAPITRE 4 : PROPRIÉTÉS LES PLUS COURANTES

color : couleur du texte
background-color : couleur de fond
font-size : taille du texte
font-family : police d’écriture
text-align : alignement du texte
margin : espace extérieur
padding : espace intérieur
border : contour de l’élément

h1{
  color:blue;
  font-size:40px;
}

div{
  padding:20px;
  border:1px solid black;
}

📘 CHAPITRE 5 : RÈGLES ET BONNES PRATIQUES

1. Toujours respecter la syntaxe CSS.
2. Mettre un point-virgule à la fin de chaque propriété.
3. Organiser le code avec des indentations.
4. Utiliser de préférence un fichier CSS externe.
5. Donner des noms clairs aux classes et aux identifiants.

Ordre de priorité :

✅ CSS en ligne
✅ CSS interne
✅ CSS externe

Le style en ligne est prioritaire sur les autres.

Conclusion : CSS complète HTML. Ensemble, ils permettent de créer des pages web modernes, professionnelles et agréables à utiliser.
