📘 CHAPITRE 1 : PRÉSENTATION DU HTML

Définition

HTML est l’acronyme de HyperText Markup Language, ce qui signifie Langage de Balisage Hypertexte.

- C’est le langage standard utilisé pour créer et structurer le contenu des pages web.
- Ce n’est pas un langage de programmation : il ne fait pas de calcul, il décrit seulement la disposition et le sens des éléments.
- Son rôle : former le squelette de la page ; la décoration (couleurs, formes) se fera plus tard avec le langage CSS.

📘 CHAPITRE 2 : LES BALISES ET LES ATTRIBUTS

2.1 Définition d’une balise

Une balise est une instruction écrite entre les symboles < et >. Elle indique au navigateur quel type d’élément afficher.

2.2 Types de balises

- Balise avec contenu : S’ouvre et se ferme, elle encadre un texte ou un autre élément.

<p>Mon texte</p>

- Balise auto-fermante : N’a pas de contenu, elle fonctionne seule.

<br>
<img>

2.3 Définition d’un attribut

Un attribut est une information complémentaire ajoutée dans la balise pour préciser son fonctionnement.

<a href="page.html">Lien</a>

Dans cet exemple :
- href est l’attribut
- page.html est sa valeur

📘 CHAPITRE 3 : STRUCTURE GÉNÉRALE D’UNE PAGE HTML

<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <title>Titre de ma page</title>
</head>
<body>
  Ici tout le contenu visible
</body>
</html>

Définition des éléments :

- <!DOCTYPE html> : précise que le document utilise HTML5.
- <html> : balise racine qui contient toute la page.
- <head> : contient les informations invisibles pour l'utilisateur.
- <meta charset="UTF-8"> : permet d'afficher correctement les accents et caractères spéciaux.
- <title> : définit le nom affiché dans l’onglet du navigateur.
- <body> : contient tout ce que l’utilisateur voit à l’écran.

📘 CHAPITRE 4 : PRINCIPALES BALISES À UTILISER

<h1> à <h6> : Balises de titre du plus grand au plus petit.

<h1>Bienvenue</h1>

<p> : Balise de paragraphe.

<p>Ceci est un paragraphe.</p>

<strong> : Met le texte en gras.

<strong>Important</strong>

<em> : Met le texte en italique.

<em>À retenir</em>

<a> : Crée un lien.

<a href="contact.html">Contact</a>

<img> : Affiche une image.

<img src="photo.jpg" alt="Photo">

<ul> et <li> : Liste à puces.

<ul> <li>Point 1</li> </ul>

<ol> et <li> : Liste numérotée.

<ol> <li>Étape 1</li> </ol>

<!-- --> : Commentaire invisible.

📘 CHAPITRE 5 : RÈGLES DE BONNE ÉCRITURE

1. Toujours fermer les balises.

✅ <p>Texte</p>
❌ <p>Texte

2. Bien imbriquer les balises.

✅ <p><b>Bonjour</b></p>
❌ <p><b>Bonjour</p></b>

3. Mettre les attributs entre guillemets.

✅ href="page.html"
❌ href=page.html

4. Écrire les balises en minuscules pour une meilleure lisibilité.

✅ Conclusion : HTML est simple, il suffit de bien connaître les balises et de respecter la structure pour construire des pages web fonctionnelles.
