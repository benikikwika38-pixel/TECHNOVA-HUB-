🌐 Formation HTML & CSS

Apprenez à créer des sites web modernes étape par étape.

📖 Leçon 1 : Introduction au HTML

HTML signifie HyperText Markup Language. C'est le langage utilisé pour créer la structure d'une page web.

<h1>Bonjour TechNova Hub</h1>

📖 Leçon 2 : Les balises HTML

Les balises permettent d'ajouter du contenu : Titres, paragraphes, images, liens, tableaux, etc.

<h1>Titre</h1> <p>Paragraphe</p> <a href="">Lien</a>

📖 Leçon 3 : Introduction au CSS

CSS permet de rendre les pages web belles et modernes.

h1{ color:blue; font-size:40px; }
