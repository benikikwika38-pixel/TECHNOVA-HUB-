🟨 DÉBUT DU COURS JAVASCRIPT

Niveau : Débutant
Prérequis : HTML et CSS

📘 CHAPITRE 1 : PRÉSENTATION DU JAVASCRIPT

Définition

JavaScript est un langage de programmation utilisé pour rendre les pages web interactives.

- Il permet d'ajouter des animations.
- Il permet de créer des boutons intelligents.
- Il permet de vérifier les formulaires.
- Il permet de modifier une page web sans la recharger.
- Il fonctionne avec HTML et CSS.

HTML crée la structure.
CSS crée le design.
JavaScript ajoute l'interactivité.

📘 CHAPITRE 2 : COMMENT AJOUTER JAVASCRIPT À UNE PAGE HTML

JavaScript interne

Le code JavaScript peut être placé directement dans une page HTML avec la balise <script>.

<script>
alert("Bienvenue sur TechNova Hub");
</script>

JavaScript externe

Le code JavaScript peut aussi être placé dans un fichier séparé ayant l'extension .js.

<script src="script.js"></script>

📘 CHAPITRE 3 : LES VARIABLES ET LES TYPES DE DONNÉES

Une variable sert à stocker une information.

Les principaux types de données sont :

  • Texte (String)
  • Nombre (Number)
  • Vrai/Faux (Boolean)
let nom = "Béni";
let age = 20;
let etudiant = true;

Pour afficher une variable dans la console :

console.log(nom);

📘 CHAPITRE 4 : LES OPÉRATEURS ET LES CONDITIONS

Les conditions permettent de prendre des décisions dans un programme.

let age = 18;

if(age >= 18){
  console.log("Majeur");
}else{
  console.log("Mineur");
}

Les principaux opérateurs sont :

  • +
  • -
  • *
  • /
  • ==
  • ===
  • !=
  • >
  • <
  • >=
  • <=

📘 CHAPITRE 5 : LES FONCTIONS

Une fonction est un bloc de code qui exécute une tâche précise.

function saluer(nom){
  alert("Bonjour " + nom);
}

saluer("Béni");

Les fonctions permettent d'éviter de répéter plusieurs fois le même code.

Une fonction peut recevoir un ou plusieurs paramètres et peut retourner un résultat grâce au mot-clé return.
